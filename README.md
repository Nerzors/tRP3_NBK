This module ist for Total RP 3 enables NerzorsBlacklistKeeper's tooltip functionality to work with TRP3's custom tooltip.

This addon requires both Total RP 3 and NerzorsBlacklistKeeper to be installed.

## ADDON - Core Information

| Property | Value |
|----------|-------|
| Author | Veplo |

| Language | Multi-Language |
| Game Version | WoW Retail, Classic Era and Classic MoP |
| Current Branch | 1.0.0 |
